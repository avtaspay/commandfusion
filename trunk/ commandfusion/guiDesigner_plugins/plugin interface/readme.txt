CommandFusion guiDesigner Plugin Development Readme
===================================================
To develop plugins for CommandFusion guiDesigner, simply create a 'Class Library' project in your favourite .NET language (C#, VB.NET, etc)
and add a reference to CFPlugin.dll. Your main class must implement the CFPlugin class.

For more information and examples, please see the CommandFusion Wiki:
http://www.commandfusion.com/wiki/index.php?title=GuiDesigner:Plugins
